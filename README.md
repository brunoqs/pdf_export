# PDF Export Redmine

With this plugin you can export all issues with all detailed information

## Installation

Works on Redmine 3.4.* 

```sh
$ cd /opt/redmine/plugins/

$ git clone https://github.com/brunoqs/pdf_export_redmine.git

$ sudo service apache2 restart
```
